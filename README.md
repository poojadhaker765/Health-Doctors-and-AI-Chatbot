# 🏥 Health Doctors and AI Chatbot

## 📌 Project Overview
This is a web application deployed on AWS EC2 using Windows Server and IIS Web Server.

It helps users explore different types of doctors and includes a simple chatbot interface.

## 🚀 Features
- Doctor categories (skin, cancer, headache, etc.)
- Simple UI
- Hosted on AWS EC2
- IIS Web Server setup

## 🛠️ Technologies Used
- HTML, CSS, JavaScript
- AWS EC2 (Windows)
- IIS Web Server

## 🌐 Deployment
Deployed using AWS EC2 with public IP.

## 📚 Learning Outcomes
- Learned EC2 setup
- IIS configuration
- Cloud deployment basics
